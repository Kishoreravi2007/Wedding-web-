# 🚀 Wedding Website Deployment Guide

## Environment Variables

Create these in your hosting platform:

```env
VITE_APP_TITLE=Wedding App
VITE_APP_DESCRIPTION=A beautiful wedding app for sharing memories
VITE_BACKEND_URL=YOUR_RENDER_BACKEND_URL
VITE_WISHES_PASSWORD=wedding
```

## Post-Deployment

1. **Test Your Website:**
   - Visit your deployed URL
   - Test photo galleries
   - Test upload functionality
   - Test on mobile devices

2. **Share with Guests:**
   - Send the URL to wedding guests
   - Share photographer portal access
   - Provide admin credentials

3. **Monitor Usage:**
   - Check hosting platform analytics
   - Monitor photo uploads
   - Track user engagement

## Maintenance

- **Regular Updates:** Push changes to GitHub for automatic deployment
- **Backup Photos:** Ensure photos are backed up regularly
- **Monitor Performance:** Check loading times and user experience
- **Update Content:** Keep wedding information current

Your wedding website is ready to go live! 🎉
