# 🎭 Photographer Portal Documentation

## 📸 Professional Wedding Photography Management System

### **🎯 Overview**
The Photographer Portal is a comprehensive tool designed for wedding photographers to professionally manage, upload, and organize wedding photos with advanced features like face tagging and guest photo discovery.

---

## 🚀 **Getting Started**

### **1. Access the Portal**
```
URL: http://localhost:8080/photographer-login
```
- Click "Photographer Portal" on the main wedding site
- Or navigate directly to `/photographer-login`

### **2. Login Credentials**
- Use the authentication system provided
- Contact admin for photographer account setup
- Supports secure role-based access

---

## 📊 **Dashboard Overview**

### **📈 Statistics Dashboard**
- **Total Photos**: Complete photo count
- **Uploaded Today**: Daily upload metrics
- **Total Views**: Guest interaction tracking
- **Average Rating**: Photo quality feedback

### **🎯 Main Navigation Tabs**

#### **1. 📤 Upload Photos Tab**
Professional photo upload interface with:
- **Drag & Drop**: Intuitive file selection
- **Batch Upload**: Multiple photo uploads
- **Metadata Forms**: Title, description, event type
- **Progress Tracking**: Real-time upload status
- **Preview System**: Photo preview before upload

#### **2. 🖼️ Photo Gallery Tab**
Complete photo management with:
- **Face Tagging**: Tag people in photos
- **Search & Filter**: Advanced photo discovery
- **Edit Metadata**: Update photo information
- **Bulk Operations**: Manage multiple photos

#### **3. 👥 People Management Tab**
Attendee database management:
- **Add/Edit People**: Complete CRUD operations
- **Role Assignment**: Bride, Groom, Family, Friends
- **Face Association**: Link people to photo tags

## 📤 **Photo Upload System**

### **🎨 Upload Interface Features**

#### **Drag & Drop Zone**
- Visual feedback during drag operations
- Multiple file selection support
- Automatic image filtering (JPG, PNG, WebP)

#### **Photo Preview Grid**
- Thumbnail previews of selected photos
- Individual photo management
- Remove unwanted photos before upload

#### **Metadata Collection**
For each photo, add:
- **Title**: Photo name/description
- **Description**: Detailed information
- **Event Type**: Ceremony type (Wedding, Reception, etc.)
- **Tags**: Custom tags for organization

### **📊 Upload Process**

1. **Select Photos**: Drag & drop or browse files
2. **Add Metadata**: Fill in photo details
3. **Review**: Check all information
4. **Upload**: Batch upload with progress tracking
5. **Confirmation**: Success notifications

## 📋 **Workflow Guide**

### **📸 Wedding Day Workflow**

#### **Pre-Event Setup**
1. **Login to Portal**: Access photographer dashboard
2. **Review People List**: Ensure all attendees are added
3. **Test Upload**: Verify upload functionality
4. **Prepare Equipment**: Camera and backup devices

#### **During Event**
1. **Capture Photos**: Take high-quality wedding photos
2. **Quick Upload**: Upload photos throughout the event
3. **Add Metadata**: Tag important moments and people
4. **Monitor Progress**: Track upload status

#### **Post-Event**
1. **Final Upload**: Upload remaining photos
2. **Face Tagging**: Tag all people in photos
3. **Review Gallery**: Ensure all photos are properly organized
4. **Share Access**: Confirm guests can access their photos

## 🚨 **Troubleshooting**

### **📤 Upload Issues**
- **File Size**: Check file size limits

- **Network Connection**: Verify stable internet
- **Browser Compatibility**: Use modern browsers

### **🔍 Search Problems**
- **No Results**: Check spelling and search terms
- **Face Tags**: Ensure people are properly tagged
- **Metadata**: Verify photo information is complete

### **👥 People Management**
- **Duplicate Names**: Avoid duplicate person entries
- **Role Assignment**: Use correct role categories
- **Database Sync**: Ensure changes are saved
